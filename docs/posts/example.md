# Markdown 常用语法示例

这里列出了一些最常用的 Markdown 语法，按日常使用频率从高到低排序。

## 1. 标题 (Headers)

使用 `#` 号的数量来表示标题级别。

**语法：**
```markdown
# 一级标题
## 二级标题
### 三级标题
```

**效果：**
（见本文档各级标题）

---

## 2. 文本样式 (Text Styling)

用于强调文本重点。

**语法：**
```markdown
**这是粗体**
*这是斜体*
~~这是删除线~~
```

**效果：**
**这是粗体**
*这是斜体*
~~这是删除线~~

---

## 3. 列表 (Lists)

用于条理清晰地排列内容。

**语法：**
```markdown
**无序列表**：
- 项目 1
- 项目 2
  - 子项目 A

**有序列表**：
1. 第一步
2. 第二步
```

**效果：**
- 项目 1
- 项目 2
  - 子项目 A

1. 第一步
2. 第二步

---

## 4. 链接与图片 (Links & Images)

注意：图片语法比链接前面多一个 `!` 号。

**语法：**
```markdown
[这是一个链接](https://www.google.com)

![这是一个图片](/images/example.png)
```

**效果：**
[这是一个链接](https://www.google.com)
*(图片需存在才能显示)*

---

## 5. 代码引用 (Code)

用于展示代码片段。

**语法：**

行内代码使用单反引号 \` ：
```markdown
请使用 `print("Hello")` 输出。
```

多行代码块使用三个反引号 \`\`\` ：
````markdown
```python
def hello():
    print("Hello World")
```
````

**效果：**
请使用 `print("Hello")` 输出。

```python
def hello():
    print("Hello World")
```

---

## 6. 引用 (Blockquotes)

用于引用他人的话或强调某段说明。

**语法：**
```markdown
> 这是一个引用段落。
> 可以换行。
```

**效果：**
> 这是一个引用段落。
> 可以换行。
